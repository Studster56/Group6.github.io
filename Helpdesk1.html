How to Install Windows (Step-by-Step)

A Beginner-Friendly Guide

Before You Begin

  • A Windows installation USB drive or DVD
  • A valid Windows product key
  • Back up your files, just in case!

Step 1: Insert the Installation Media

  • Plug in the USB or insert the DVD
  • Restart your computer

Step 2: Enter the Boot Menu

  • At startup, press the key to open the Boot Menu (e.g., Esc, Del, F12)
  • Select your USB or DVD as the boot device
  • Use arrow or function keys to move your boot device to the top
  • Press F10 (or listed key) to Save & Exit

Step 3: Start Windows Setup

  • You’ll see a "Windows Setup" screen
  • Select language, time, and keyboard
  • Click Next, then Install Now

Step 4: Enter Product Key

  • Enter your Windows product key
  • Or click “I don’t have a product key” to enter it later

Step 5: Choose Installation Type

  • Choose Custom: Install Windows only (advanced)

Step 6: Choose a Drive

  • Select the main partition (your main drive)
  • Click Format for a clean install (erases everything)
  • Click Next

Step 7: Let Windows Install

  • Installation takes ~20–30 minutes
  • Your PC may restart multiple times (this is normal)

Step 8: Set Up Windows

  • Choose your region and keyboard layout
  • Connect to Wi-Fi
  • Create or sign in with a Microsoft account
  • Choose privacy settings

Step 9: Install Updates and Drivers

  • Let Windows check for updates and drivers
  • Make sure you're connected to the internet

Employee Training Manual – Windows Installation Basics

As a new IT support team member, it’s important to understand the essentials of Windows installation.

  • Always check system requirements (RAM, storage, CPU compatibility)
  • Use the Windows Media Creation Tool for bootable USBs
  • Clean Install: Erases everything and installs Windows from scratch
  • Upgrade: Keeps files and apps, used for system updates
  • Backup data before beginning
  • Install all updates and drivers after setup
  • Communicate clearly and calmly with users during support
